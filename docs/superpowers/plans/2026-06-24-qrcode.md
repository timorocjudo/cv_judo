# QR Code par profil judoka — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ajouter la génération d'un QR code PNG par profil judoka, accessible via `/api/qrcode/[slug]`, affiché dans le HeroBlock (profils publics) et dans le dashboard (profils public + private).

**Architecture:** Route API Next.js `/api/qrcode/[slug]/route.ts` génère un PNG 512×512 via `qrcode.toBuffer()`. Le composant `QRCodeDisplay` affiche via `<img src>` et propose le téléchargement. HeroBlock et la page dashboard consomment ce composant.

**Tech Stack:** Next.js 14 App Router, `qrcode` + `@types/qrcode`, Supabase (RLS), Tailwind CSS, Vitest

## Global Constraints

- Couleur QR : dark `#1B3A6B`, light `#FFFFFF`
- Taille QR : 512×512px, margin 2, errorCorrectionLevel `'M'`
- URL encodée : `${NEXT_PUBLIC_SITE_URL}/${slug}` (fallback `https://ipponid.com`)
- Cache-Control : `public, max-age=31536000, immutable`
- Nom de fichier téléchargé : `ipponid-[slug].png`
- API retourne 404 pour `draft` et slug inconnu ; 200 pour `public` et `private`
- Toggle QR dans HeroBlock : uniquement si `visibility === 'public'`
- Section QR dans dashboard : uniquement si `visibility !== 'draft'`

- [ ] **Step 5 : Créer la route API**

Créer `app/api/qrcode/[slug]/route.ts` :

```typescript
import { NextRequest } from 'next/server'
import QRCode from 'qrcode'
import { createClient } from '@/lib/supabase/server'

export async function GET(
  _req: NextRequest,
  { params }: { params: { slug: string } }
) {
  const supabase = createClient()
  const { data } = await supabase
    .from('profiles')
    .select('slug, visibility')
    .eq('slug', params.slug)
    .maybeSingle()

  if (!data || data.visibility === 'draft') {
    return new Response(null, { status: 404 })
  }

  const siteUrl = process.env.NEXT_PUBLIC_SITE_URL ?? 'https://ipponid.com'
  const profileUrl = `${siteUrl}/${params.slug}`

  const buffer = await QRCode.toBuffer(profileUrl, {
    width: 512,
    margin: 2,
    color: { dark: '#1B3A6B', light: '#FFFFFF' },
    errorCorrectionLevel: 'M',
  })

  return new Response(buffer, {
    status: 200,
    headers: {
      'Content-Type': 'image/png',
      'Cache-Control': 'public, max-age=31536000, immutable',
    },
  })
}
```

Créer `components/QRCodeDisplay.tsx` :

```typescript
'use client'

import { useState } from 'react'

interface QRCodeDisplayProps {
  slug: string
  size?: number
  showLabel?: boolean
}

export default function QRCodeDisplay({ slug, size = 200, showLabel = true }: QRCodeDisplayProps) {
  const [isDownloading, setIsDownloading] = useState(false)

  async function handleDownload() {
    setIsDownloading(true)
    try {
      const res = await fetch(`/api/qrcode/${slug}`)
      const blob = await res.blob()
      const blobUrl = URL.createObjectURL(blob)
      const a = document.createElement('a')
      a.href = blobUrl
      a.download = `ipponid-${slug}.png`
      document.body.appendChild(a)
      a.click()
      document.body.removeChild(a)
      URL.revokeObjectURL(blobUrl)
    } finally {
      setIsDownloading(false)
    }
  }

  return (
    <div className="bg-white rounded-xl border border-outline-variant shadow-sm p-4 inline-flex flex-col items-center gap-3">
      {showLabel && (
        <p className="font-montserrat font-bold text-primary text-xs uppercase tracking-wide">
          Mon QR Code IpponId
        </p>
      )}
      <img
        src={`/api/qrcode/${slug}`}
        alt={`QR code du profil IpponId de ${slug}`}
        width={size}
        height={size}
        className="rounded-lg"
      />
      <p className="text-xs text-on-surface-variant">ipponid.com/{slug}</p>
      <button
        onClick={handleDownload}
        disabled={isDownloading}
        className="flex items-center gap-2 border border-outline-variant text-on-surface text-xs font-bold uppercase tracking-wider px-4 py-2 rounded-full hover:bg-surface-container transition-colors disabled:opacity-60 disabled:cursor-not-allowed"
      >
        {isDownloading ? 'Téléchargement…' : 'Télécharger en PNG'}
      </button>
    </div>
  )
}
```
